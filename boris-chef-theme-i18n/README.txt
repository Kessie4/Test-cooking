
Shopify Theme — Boris Chef (DE/FR/EN)
=====================================
• Enthält Startseite (Abschnitte) + Reservierungsseite (Formular).
• Sprachen: DE, FR, EN (DE empfohlen als Shop-Standardsprache).

Installation (ZIP Upload)
-------------------------
1) Online Store → Themes → Add theme → Upload zip → wähle diese Datei.
2) Erstelle Seite: Online Store → Pages → Add page → Title „Reservieren“ → Theme template = `page.reserver` → Save.
3) Navigation → füge Link zu `/pages/reserver` hinzu.
4) Customize → Bilder/Text/Preise anpassen.

GitHub-Methode
--------------
1) Erstelle Repository, lade alle Dateien (ohne zusätzlichen Oberordner) hoch.
2) Online Store → Themes → Add theme → Connect from GitHub → Repo wählen.
3) Publish oder Preview.

Hinweise
--------
• Dieses Theme bringt EN/FR/DE-Locales mit. Die Shop-Standardsprache stellst du in Shopify Admin → Languages.
• Bilder in Abschnitten konfigurierbar; keine Speisenfotos eingebettet.
